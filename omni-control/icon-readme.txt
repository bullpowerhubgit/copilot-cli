# Icon-Erstellung für Omni Control Suite Installer
#
# Um ein professionelles Icon für den Installer zu erstellen:
#
# 1. Verwenden Sie ein Icon-Design-Tool wie:
#    - https://favicon.io/favicon-generator/
#    - https://www.icoconverter.com/
#    - Adobe Illustrator oder GIMP
#
# 2. Erstellen Sie ein Icon mit folgenden Eigenschaften:
#    - Format: ICO (mehrere Größen: 16x16, 32x32, 48x48, 256x256)
#    - Design: Moderne, technische Darstellung (z.B. Monitor/Screen mit Verbindungslinien)
#    - Farben: Professionelle Farbpalette (Blau/Grau für Tech-Design)
#
# 3. Speichern Sie die icon.ico Datei im omni-control Verzeichnis
#
# 4. Alternativ können Sie ein vorhandenes Icon verwenden:
#    - Windows System Icons: shell32.dll, imageres.dll
#    - Beispiel: icon.ico aus Windows Systemordner kopieren
#
# Hinweis: Für die Erstinstallation kann ein einfaches generisches Icon verwendet werden.
